# AI Persona

A modern, fluid AI persona chatbot with a FastAPI backend and a Next.js frontend.

## 🚀 Features
- **Persona-based Chat**: Multiple AI personas with unique traits.
- **Fluid UI**: Smooth animations using Framer Motion.
- **Real-time Streaming**: Server-Sent Events (SSE) for instant AI responses.
- **Glassmorphism Design**: Sleek and modern aesthetic.

## 🛠 Tech Stack
- **Frontend**: Next.js 15, React, Tailwind CSS, Framer Motion.
- **Backend**: FastAPI, Python, Google Gemini API.

## 🏃 Getting Started

### Backend
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Run the development server:
   ```bash
   uv run fastapi dev
   ```

### Frontend
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
